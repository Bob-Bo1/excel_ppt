# Excel 数据透视自动生成 PPT 图表工具 — 产品需求文档

> 版本 3.0 | 最后更新：2026-04-10

---

## 1. 项目概述

本工具旨在将 Excel 中的原始业务数据，通过**配置驱动**的方式自动完成数据透视，并将结果以图表形式批量生成到 PowerPoint 演示文稿中。用户无需编写代码或手动操作 PPT，只需准备好原始数据和一份配置表，即可一键生成包含多页、多图表的专业 PPT。

## 2. 目标用户

| 角色 | 场景 |
|------|------|
| 业务分析师 | 定期制作经营分析报告，需要从 Excel 数据生成大量 PPT 图表 |
| 项目经理 | 周报/月报中需要展示项目数据的多维透视 |
| 销售运营 | 按区域、产品、时间等维度汇总销售数据并汇报 |
| 数据运营 | 需要定期将数据可视化并分发给非技术人员 |

## 3. 核心功能

### 3.1 数据读取
- 支持读取 `.xlsx` / `.xls` 格式的 Excel 原始数据
- 数据要求第一行为表头，后续行为数据记录
- 自动识别所有列名，用于后续透视配置

### 3.2 配置驱动的数据透视
- 通过独立的 `config.xlsx` 文件描述透视规则和图表需求
- 每一行配置对应一张图表
- 同一「页面编号」的多行配置会被合并到同一张幻灯片上
- 支持的聚合方式：`sum`（求和）、`mean`（平均）、`count`（计数）

### 3.3 图表生成
支持三种图表类型：

| 图表类型 | 说明 |
|---------|------|
| **柱形图** | 簇状柱形图，适合比较不同类别的数值 |
| **折线图** | 标准折线图，适合展示趋势变化 |
| **组合图** | 柱形 + 折线组合，适合在同一图表中展示不同量纲的数据（如销售额柱形 + 利润折线） |

### 3.4 自动布局
- 根据每页图表数量自动计算图表的位置和尺寸
- 支持 1~9 张图表的自适应网格布局：

| 图表数量 | 布局方式 |
|---------|---------|
| 1 | 单图全屏 |
| 2 | 1 行 × 2 列 |
| 3~4 | 2 行 × 2 列 |
| 5~6 | 2 行 × 3 列 |
| 7~9 | 3 行 × 3 列 |

### 3.5 PPT 模板支持
- 可选择已有的 `.pptx` 模板文件作为基础
- 留空则自动创建 10×7.5 英寸的空白 PPT
- 每页自动添加标题文本框

### 3.6 GUI 交互
- 提供基于 Tkinter 的桌面图形界面
- 文件选择对话框（原始数据、配置文件、PPT 模板）
- 实时运行日志显示处理进度
- 一键打开输出文件夹

## 5. 配置文件规范（config.xlsx）

配置文件为标准 Excel 文件，第一行为表头，每一行描述一张图表的生成规则。

| 列名 | 类型 | 必填 | 说明 | 示例 |
|------|------|------|------|------|
| 页面编号 | 数字 | 是 | 同一编号的配置行放在同一页 PPT | `1` |
| 页面标题 | 文本 | 是 | 幻灯片顶部标题文字 | `销售概览` |
| 透视行 | 文本 | 否 | X 轴分组维度（对应 pivot_table 的 index） | `月份` |
| 透视列 | 文本 | 否 | 系列展开维度（对应 pivot_table 的 columns） | `区域` |
| 透视值 | 文本 | 是 | 需要聚合的数值字段，多个用英文逗号分隔 | `销售额,利润` |
| 计算方式 | 文本 | 是 | 聚合函数，不区分大小写 | `sum` / `mean` / `count` |
| 图表类型 | 文本 | 是 | 图表展现形式 | `柱形图` / `折线图` / `组合图` |
| 柱形列 | 文本 | 组合图必填 | 组合图中显示为柱形的字段名 | `销售额` |
| 折线列 | 文本 | 组合图必填 | 组合图中显示为折线的字段名 | `利润` |

### 配置示例

| 页面编号 | 页面标题 | 透视行 | 透视列 | 透视值 | 计算方式 | 图表类型 | 柱形列 | 折线列 |
|---------|---------|--------|--------|--------|---------|---------|--------|--------|
| 1 | 销售概览 | 月份 | 区域 | 销售额 | sum | 柱形图 | | |
| 1 | 销售概览 | 月份 | | 销售额,利润 | sum | 组合图 | 销售额 | 利润 |
| 2 | 利润分析 | 月份 | 区域 | 利润 | mean | 折线图 | | |

上述配置将生成 2 页 PPT：
- 第 1 页「销售概览」包含 2 张图表（左：按区域分组的月度销售额柱形图；右：销售额+利润的组合图）
- 第 2 页「利润分析」包含 1 张折线图

## 6. 输出规范

- 文件名格式：`自动生成PPT_YYYYMMDD_HHMMSS.pptx`
- 输出目录：程序所在目录
- 幻灯片尺寸：10 × 7.5 英寸（4:3），使用模板时沿用模板尺寸

## 7. 技术架构

```
┌─────────────────────────────────────────┐
│              GUI 层 (Tkinter)            │
│  文件选择 | 按钮操作 | 日志展示           │
├─────────────────────────────────────────┤
│           业务逻辑层 (Python)             │
│  配置解析 | 数据透视 | 图表生成 | 布局计算  │
├─────────────────────────────────────────┤
│              依赖库层                     │
│  pandas | python-pptx | openpyxl | lxml  │
└─────────────────────────────────────────┘
```

### 线程模型
- **主线程**：运行 Tkinter 事件循环，处理 UI 交互
- **工作线程**：执行 PPT 生成任务，通过消息队列将 UI 更新调度回主线程，避免界面卡死和线程安全问题

## 10. 已知限制与后续规划

### 当前版本限制
- 仅支持柱形图、折线图、组合图三种图表类型
- 组合图中柱形和折线共享同一 Y 轴，暂不支持双 Y 轴
- 布局算法为固定网格，不支持自定义图表位置和尺寸
- 图表样式使用 PPT 默认样式，暂不支持自定义颜色/字体

### 后续可扩展方向
- 支持更多图表类型（饼图、堆叠柱形图、面积图等）
- 支持双 Y 轴（组合图中折线使用次坐标轴）
- 支持在配置中自定义图表位置、尺寸、颜色方案
- 支持数据筛选条件（在透视前过滤数据）
- 支持图表标题、数据标签等细节配置
- 导出为 PDF
